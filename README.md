## 安装

```
npm install rty-vue-ui -S
```

### 引入

``` js
//main.js
import 'rty-vue-ui/dist/rty-vue-ui.min.css'
import rtyVueUi from 'rty-vue-ui'
Vue.use(rtyVueUi)
```

### 使用

``` html
<rty-button></rty-button>
```
